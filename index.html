Sou Bernardo Puppim

formado em Engenharia Mecânica pela Universidade Federal de Ouro Preto, possuo MBA em Gestão de Projetos pela USP - Escola Superior de Agricultura “Luiz de Queiroz” e atualmente sou estudante especial no Mestrado em Sistemas Computacionais pela Universidade Federal Fluminense. Gosto de me manter estudando para aumentar meu repertório e poder lidar de forma criativa e mais eficiente com os desafios de minha caminhada.

Após cinco anos atuando na área de Engenharia Mecânica na indústria de Petróleo e Gás, decidi me reinventar e migrar para ciência de dados. Por enxergar nessa área a possibilidade de unir as capacidades analíticas que desenvolvi quanto engenheiro e as novas tecnologias que vêm redesenhando a forma como tomamos decisões.

Como Cientista de Dados venho trabalhando em projetos pessoais para adquirir a experiência exigida pelo mercado, referente à análise e solução de problemas de negócios com base em dados.

Meu próximo passo é conseguir uma oportunidade para aplicar meus conhecimentos dentro de uma empresa e facilitar as tomadas de decisão e planejamentos estratégicos com a utilização inteligente dos dados disponíveis.

Cursos e Certificados

Statistics with Python

Programa de cursos integrados oferecidos por University of Michigan pela plataforma de ensino Coursera. Este programa contém 3 cursos: Entendendo e Visualizando Dados com Python, Análise Estatística Inferencial com Python e Ajustando modelos estatísticos a dados com Python. O programa aborda como utilizar dados para estimar e avaliar teorias, construir intervalos de confiança, interpretar resultados inferenciais e aplicar procedimentos de modelagem estatística mais avançados.

Habilidades desenvolvidas

  • Python, Pandas, Numpy, Matplotlib, Statsmodels.Api
  • Análise, Visualização e manipulação de dados
  • Modelos Estatísticos, Estatísticas, Métodos de inferência
  • Intervalo de confiança, Teste de hipótese, regressão estatística
  • Certificado

Machine Learning

Programa de cursos integrados oferecidos por DeepLearning.AI e Stanford pela plataforma de ensino Coursera. Programa composto por 3 cursos: “Aprendizado de máquina supervisionado: regressão e classificação”, “Algoritmos de Aprendizado Avançado” e “Aprendizado não supervisionado, recomendações, aprendizado por reforço”.

Habilidades desenvolvidas

  • Árvore de decisão, Conjunto de árvores,
  • Rede neural artificial, Regressão logística, Regressão linear,
  • Regressão logística para Classificação, Aprendizado Supervisionado
  • Python, Numpy, scikitlearn Tensorflow, Xgboost
  • Regularização para evirtar overfitting, gradiente descendente
  • Certificado

SQL para Ciência de Dados

Curso de linguagem SQL oferecido por University of California pela plataforma de ensino Coursera. O curso apresenta fundamentos da linguagem e como trabalhar com os dados para fins de ciência de dados. Aborda como realizar consultas simples e complexas em banco de dados para limitar os resultados, criar tabelas, aplicar operadores e combinar dados, etc.

Habilidades desenvolvidas

  • SQL
  • Sqlite
  • Certificado

Data Science e Big Data:
Decisões Baseadas em Dados

Curso on-line do program xPRO ofertado pelo Massachusetts Institute of Technology, conhecido mundialmente como MIT. O curso apresenta o estado da arte da ciência de dados e big data, aborda teoria e a prática relacionadas a sistemas de recomendação, regressões, modelos de rede e gráficos, detecção de anomalias, testes de hipótese, aprendizagem de máquina e análise de big data.

  • Certificado
