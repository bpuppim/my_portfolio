Cursos e Certificados

Statistics with Python

Programa de cursos integrados oferecidos por University of Michigan pela plataforma de ensino Coursera. Este programa contém 3 cursos: Entendendo e Visualizando Dados com Python, Análise Estatística Inferencial com Python e Ajustando modelos estatísticos a dados com Python. O programa aborda como utilizar dados para estimar e avaliar teorias, construir intervalos de confiança, interpretar resultados inferenciais e aplicar procedimentos de modelagem estatística mais avançados.

Habilidades desenvolvidas

  • Python, Pandas, Numpy, Matplotlib, Statsmodels.Api
  • Análise, Visualização e manipulação de dados
  • Modelos Estatísticos, Estatísticas, Métodos de inferência
  • Intervalo de confiança, Teste de hipótese, regressão estatística
  • Visualizar Certificado

Machine Learning

Programa de cursos integrados oferecidos por DeepLearning.AI e Stanford pela plataforma de ensino Coursera. Programa composto por 3 cursos: “Aprendizado de máquina supervisionado: regressão e classificação”, “Algoritmos de Aprendizado Avançado” e “Aprendizado não supervisionado, recomendações, aprendizado por reforço”.

Habilidades desenvolvidas

  • Árvore de decisão, Conjunto de árvores,
  • Rede neural artificial, Regressão logística, Regressão linear,
  • Regressão logística para Classificação, Aprendizado Supervisionado
  • Python, Numpy, scikitlearn Tensorflow, XgboostGenericName
  • Regularização para evirtar overfitting, gradiente descendente
  • Visualizar Certificado

SQL para Ciência de Dados

Curso de linguagem SQL oferecido por University of California pela plataforma de ensino Coursera. O curso apresenta fundamentos da linguagem e como trabalhar com os dados para fins de ciência de dados. Aborda como realizar consultas simples e complexas em banco de dados para limitar os resultados, criar tabelas, aplicar operadores e combinar dados, etc.

Habilidades desenvolvidas

  • SQL
  • Sqlite
  • Visualizar Certificado
